# Проект по аренде жилья

## Цель проекта

Создать полнофункциональное back-end приложение для системы аренды жилья, включающее управление объявлениями, функциональность поиска и фильтрации по различным параметрам.

---

# Описание функционала приложения

## Управление объявлениями

### 1. Создание объявления

Пользователь вводит информацию о недвижимости:

- Заголовок
- Описание
- Местоположение
- Цена
- Количество комнат
- Тип жилья (квартира, дом и т.д.)

### 2. Редактирование объявления

Пользователь может изменить любую информацию в существующем объявлении.

### 3. Удаление объявления

Пользователь может удалить своё объявление из базы данных.

### 4. Управление доступностью объявления

Переключение статуса объявления (активно/неактивно), чтобы временно скрыть или снова сделать видимым объявление.

---

## Поиск и фильтрация

### 1. Поиск по ключевым словам

Пользователь вводит ключевые слова, по которым производится поиск в заголовках и описаниях объявлений.

### 2. Фильтрация по параметрам

- **Цена** — возможность указать минимальную и максимальную цену
- **Местоположение** — возможность указать город или район в Германии
- **Количество комнат** — возможность указать диапазон количества комнат
- **Тип жилья** — возможность выбрать тип жилья: квартира, дом, студия и т.д.

### 3. Сортировка результатов

- Возможность сортировки по цене (возрастание/убывание)
- По дате добавления (новые/старые)

---

## Пользовательская аутентификация и авторизация

### 1. Регистрация пользователя

Пользователь вводит свои данные для создания аккаунта:

- Имя
- Email
- Пароль

### 2. Вход в систему

Ввод email и пароля для доступа к аккаунту.

### 3. Разграничение прав доступа

- **Арендатор** — может просматривать и фильтровать объявления
- **Арендодатель** — может создавать, редактировать и удалять свои объявления

---

## Бронирование

### 1. Создание бронирования

Пользователь может забронировать жилье на определённые даты.

### 2. Просмотр бронирований

Пользователь может просматривать свои активные и завершённые бронирования.

### 3. Отмена бронирования

Пользователь может отменить бронирование до определённой даты.

### 4. Подтверждение бронирования

Арендодатель может подтверждать или отклонять запросы на бронирование.

---

## Рейтинги и отзывы

### 1. Оставление отзыва

Пользователь, который снял жильё, может оставить отзыв и рейтинг для конкретного объявления.

### 2. Просмотр отзывов

Возможность просмотра всех отзывов для конкретного объявления.

---

## Дополнительный функционал

### 1. Сортировка по популярности

По количеству просмотров или отзывов.

### 2. История поиска

- Сохранение ключевых слов поиска — запись ключевых слов, по которым пользователь осуществлял поиск, в отдельную таблицу
- Вывод популярных запросов — запросы по наиболее частым ключевым словам выводятся первыми

### 3. История просмотров объявлений

- Сохранение просмотров — запись информации о каждом просмотре конкретного объявления пользователем
- Вывод популярных объявлений — объявления с наибольшим количеством просмотров выводятся первыми

---

## Технические требования

- Django — использовать Django для разработки основной логики приложения, управления базами данных и создания API
- MySQL — основная база данных для хранения данных об объявлениях и пользователях
